Get the correct type of value from stdin

Template Parameters
TThe expected return type
Parameters
promptA short prompt message printed on stdout, which will appear before user input
retryThe flag for whether to retry the whole process when an input exception occurs, whether it's the type mismatch or invalid input
Returns
If retry flag is set to false and an exception does occurs, the function clears any invalid states, and return a default-constructed object of the expected type. Otherwise return the correct input.
